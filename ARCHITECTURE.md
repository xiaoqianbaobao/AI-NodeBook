# AI笔记本系统架构

## 系统架构图

```
┌─────────────────────────────────────────────────────────────┐
│                    AI笔记本应用程序                          │
├─────────────────────────────────────────────────────────────┤
│  会议纪要模块  │  笔记管理模块  │  学习训练模块  │  系统设置模块  │
├─────────────────────────────────────────────────────────────┤
│  语音录制     │  PDF上传      │  闪卡管理     │  API配置      │
│  语音转文字   │  视频处理     │  出题训练     │  数据管理     │
│  纪要生成     │  文本笔记     │  AI助手      │  界面设置     │
│  卡片展示     │  AI摘要      │  进度统计     │  连接测试     │

├─────────────────────────────────────────────────────────────┤
│  科大讯飞ASR API  │  DeepSeek LLM API  │  本地存储系统    │
```

## 技术栈

### 前端技术
- **React 18**: 现代化UI框架
- **TypeScript**: 类型安全
- **Material-UI**: UI组件库
- **Zustand**: 状态管理
- **Vite**: 构建工具

### 桌面应用
- **Electron**: 跨平台框架
- **IPC通信**: 进程间通信
- **文件系统**: 本地文件操作

### API服务
- **科大讯飞ASR**: 语音识别
- **DeepSeek LLM**: AI内容生成
- **本地存储**: 数据持久化

## 数据流

```
用户输入 → 前端组件 → 状态管理 → API服务 → 外部API → 数据处理 → 状态更新 → UI更新
```

## 模块关系

- **会议纪要**: 语音 → ASR → LLM → 结构化数据
- **笔记管理**: 文件/文本 → 解析 → LLM → 摘要/关键点
- **学习训练**: 数据 → 闪卡生成 → 训练系统 → 进度统计
- **系统设置**: 配置 → 验证 → 存储 → 应用

## 部署架构

```
开发环境: npm run dev → 热重载开发
构建环境: npm run build → 生产构建
运行环境: npm run electron → 桌面应用
分发环境: npm run dist:win → Windows安装包
```
