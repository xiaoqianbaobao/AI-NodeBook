# AI笔记本用户指南

## 快速开始

### 1. 安装和启动

#### 方法一：使用安装脚本（推荐）
1. 双击运行 `install.bat`
2. 等待安装完成
3. 双击运行 `start.bat` 启动应用程序

#### 方法二：手动安装
1. 确保已安装 Node.js 16.0 或更高版本
2. 打开命令行，进入项目目录
3. 运行 `npm install` 安装依赖
4. 运行 `npm run build` 构建应用程序
5. 运行 `npm run electron` 启动应用程序

### 2. 首次配置

启动应用程序后，首先需要配置API密钥：

1. 点击左侧菜单的"系统设置"
2. 在"API配置"部分输入以下密钥：

#### 科大讯飞ASR API（语音转文字）
- 访问 https://www.xfyun.cn/
- 注册账号并申请语音听写服务
- 获取API密钥并填入设置页面

#### DeepSeek LLM API（AI内容生成）
- 访问 https://platform.deepseek.com/
- 注册账号并获取API密钥
- 填入设置页面（已预填测试密钥）

3. 点击"测试"按钮验证API连接
4. 点击"保存设置"保存配置

## 功能使用指南

### 🎤 会议纪要功能

#### 录制会议
1. 点击左侧菜单的"会议纪要"
2. 点击"开始录音"按钮开始录制
3. 录制过程中可以：
   - 点击暂停/继续按钮控制录制
   - 查看录制时长
4. 点击"停止录音"结束录制

#### 查看会议纪要
- 录制结束后，系统自动生成会议纪要
- 包含：会议摘要、讨论议题、行动项
- 可以编辑、删除会议记录
- 支持导出为PDF或Word文档

### 📝 笔记管理功能

#### 上传PDF文件
1. 点击左侧菜单的"笔记管理"
2. 选择"上传文件"标签页
3. 点击"选择PDF文件"按钮
4. 选择要处理的PDF文件
5. 系统自动提取内容并生成摘要

#### 处理Bilibili视频
1. 选择"Bilibili视频"标签页
2. 粘贴视频链接
3. 点击"处理视频"按钮
4. 系统自动提取字幕并生成笔记

#### 创建文本笔记
1. 选择"文本笔记"标签页
2. 输入笔记标题和内容
3. 点击"创建笔记"按钮
4. 系统自动生成摘要和关键点

### 🧠 学习训练功能

#### 闪卡管理
1. 点击左侧菜单的"学习训练"
2. 选择"闪卡管理"标签页
3. 点击"创建闪卡"按钮
4. 输入问题和答案
5. 设置分类和难度等级

#### 出题训练
1. 选择"出题训练"标签页
2. 点击"开始训练"按钮
3. 查看问题，思考答案
4. 点击"显示答案"查看正确答案
5. 选择"答对了"或"答错了"
6. 系统记录学习进度

#### AI助手
1. 选择"AI助手"标签页
2. 点击"开始对话"按钮
3. 在对话框中输入问题
4. AI基于你的笔记内容进行智能答疑

### ⚙️ 系统设置

#### 界面设置
- 可以切换深色/浅色模式
- 调整界面主题色彩

#### 数据管理
- 查看笔记、会议、闪卡数量统计
- 导出所有数据为JSON文件
- 清空所有数据（谨慎操作）

## 使用技巧

### 提高语音识别准确率
1. 确保在安静的环境中录制
2. 说话清晰，语速适中
3. 避免背景噪音干扰
4. 录制时间不要过长（建议60分钟以内）

### 优化AI生成效果
1. 确保API密钥配置正确
2. 提供清晰、结构化的输入内容
3. 对于重要内容，可以手动编辑AI生成的结果

### 学习效率提升
1. 定期进行闪卡训练
2. 利用AI助手解答疑问
3. 及时整理和分类笔记
4. 定期回顾会议纪要

## 常见问题

### Q: 语音录制没有声音？
A: 检查麦克风权限设置，确保应用程序可以访问麦克风。

### Q: API调用失败？
A: 检查网络连接和API密钥配置，确保密钥有效且有足够的调用额度。

### Q: 应用程序启动失败？
A: 确保Node.js版本符合要求，重新安装依赖包。

### Q: 数据丢失？
A: 数据存储在本地，定期使用导出功能备份数据。

### Q: 如何更新应用程序？
A: 重新下载最新版本，或使用git pull更新代码后重新构建。
